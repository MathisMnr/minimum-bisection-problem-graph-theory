# minimum-bisection-problem-graph-theory
Etude portée sur le problème de la bipartition minimale d'un graphe. Ce repository regroupe plusieurs algorithmes de tri en C++, que nous avons testé dans leur complexité et dans leur rapidité d'exécution.  Un graph d'input nous est donné, et nous en déterminons via quatre algorithmes différents une solution (exacte, heuristique, metaheuristique..)
